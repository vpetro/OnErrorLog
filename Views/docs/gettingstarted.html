Follow these examples below to get started.

Inserting an Exception

Copy the Client Library in your application and follow this example to insert your first exception:
from Packages.onerrorlog import onerrorlog

onerrorlog_url = 'http://www.onerrorlog'
api_key = '4e62d31abc98554d11000000' #Get this from the side bar once you logged in
application_name = 'MyAppName' #You can arbitrarily choose an application name

#Instantiate the onerrorlog client
on_client = onerrorlog.OnErrorLog(onerrorlog_url, api_key, application_name)

self.on_client.send_log('This is a sample message', onerrorlog.SEVERITY_DEBUG)


Using the Client In A Web Application

The following is a sample log entry using a web application, in this case tornado.
def send_error(self, status_code, **kwargs):
    from Packages.onerrorlog import onerrorlog

    onerrorlog_url = 'http://www.onerrorlog'
    api_key = '4e62d31abc98554d11000000' #Get this from the side bar once you logged in
    application_name = 'MyAppName' #You can arbitrarily choose an application name

    #Instantiate the onerrorlog client
    on_client = onerrorlog.OnErrorLog(onerrorlog_url, api_key, application_name)

    self.on_client.send_log(kwargs['exception'], onerrorlog.SEVERITY_ERROR, filename=str(self.__class__.__name__),
        url=self.request.full_url(), status_code=status_code,
        headers=self.request.headers, parameters=self.request.arguments)


    self.finish()
